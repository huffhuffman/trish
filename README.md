## how to play

`← ↓ →` でミノ（落下してくる赤いブロック）を移動

`スペースキー`でミノを回転

`↑` で特殊能力を発動

**特殊能力の効果:** ミノを無作為に変更し、一番上に戻す

**特殊能力の制限:** 特殊能力を発動した後は、ミノを一度設置するまでは再発動できない
